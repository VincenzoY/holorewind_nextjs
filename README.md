# HoloRewind

Frontend for HoloRewind project.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Requires accompanying pocketbase deployement at URL given by `NEXT_PUBLIC_POCKETBASE_ADDRESS` env var.

## Deployment

This uses a Dockerfile for deployment. Deployed using Coolify.
